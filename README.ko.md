## 철학
- Code to Product: 코드는 실제 사용자 경험으로 연결되는 브릿지입니다.
- Compact하게: 불필요한 비즈니스 로직과 복잡도를 줄입니다.
- 레거시 패턴보다 현대적인 아키텍처를 선호합니다.
- 실용적인 디자인/구현 트레이드오프를 빠르게 판단합니다.

## Rockstar Principle
Netflix Rockstar Principle 관점을 따릅니다.
- 뛰어난 엔지니어는 코딩, 디버깅, 실행 성능 전반에서 압도적인 임팩트를 만듭니다.
- 창의성과 패턴 인식 능력이 핵심 생산성 배수라고 봅니다.

참고: [Rockstar Principle](https://www.rwaconsultants.com/news/rock-star-software-engineers-worth-their-weight-in-gold)
